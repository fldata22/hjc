# CLAUDE.md

API backend for **HJC Mission Control** — crusade director command system.

## Tech Stack

- Laravel (PHP)

## Commands

```bash
composer install         # PHP deps
php artisan serve        # Dev server (default: http://localhost:8000)
php artisan migrate      # DB migrations
php artisan test         # Run tests
```
